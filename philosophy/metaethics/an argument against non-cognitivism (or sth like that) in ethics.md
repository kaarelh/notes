suppose that our values were more like "yay x" type things, as opposed to "x is good" type things, so as opposed to propositional claims. then it seems like if you're uncertain between preference utilitarianism and hedonism, you'd be happy picking a policy which maxes some linear combo of both until the end of time **even if further reasoning would lead you to only adapt one of them** (because they don't care about the reasoning, they just want the world to be a certain way regardless, and this is what optimality looks like in that case). this seems undesirable — you wouldn't want to do that — you'd want to act according to what you arrived at.

this undesirable thing doesn't happen if you taking these different theories seriously is more like entertaining them to be correct hypotheses about what's good, instead of these prime wishings.

i guess this is generally an argument that to get responsiveness to reason of one's behavior, an antirealist might want to take ethical theories to be making predictions about reasoning or something, because otherwise critch-russell stuff would say that updating isn't good!
