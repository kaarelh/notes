
there are structures in the world. we can understand them by making analogous structures in our minds. (example: the natural numbers)

a confusing thing: plausibly, one can only start to speak of these structures after one already has some way of looking at the world (when i said "there are structures in the world", this really might need some reference conception of the world already). but then didn't one already have these structures also? well, no! this is analogous to how not everything implied by axioms is immediately given to someone that believes the axioms — one really has to think. the structure will in some sense be already there in one's existing picture of the world, but it will not be explicitly there. coming to see it explicitly can be hard thinking-work (much like it can be hard to see anything in math, like some connection between algebraic geometry and complex analysis). 

speculation: despite the dependence on prior concepts, basically all learning has this form. even while one learns one's first language, one is seeing structures in some mix of the fragment of that language one has already grasped together with some ur-language

maybe: when you talk to another person a bunch, you learn a bunch of their mind-structures because those structures are the natural ways to generate what they are saying. and now you can use these structures for your own purposes.
