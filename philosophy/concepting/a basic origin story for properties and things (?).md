
this is mostly a very standard picture, eg from ruth millikan or eliezer yudkowsky

* there are some ur-properties
	* maybe these could even be random/arbitrary in some sense?
* you have some objects
	* for now, don't ask where these came from
* imagine plotting points corresponding to these objects on axes corresponding to these ur-properties
* now find clusters (assuming some good norm). those are kinds: galaxies, stars, americans, europeans, english speakers, humans, organisms, etc
	* a kind will generally vary along some axes and not others
	* maybe each kind has a specification of some axes it can vary along. i think it will be less clear from this considered data alone which axes it cannot vary along — like, it is fine to let it vary along some axes along which variance hasn't been seen yet. eg maybe in some context you want to make artificial grass which is red and still call it that. i think resolving which variance not to allow depends on purposes (like, if making a football liquid would make it unusable, then we say we don't consider liquid things footballs)
* by looking for directions of common variance, you can discover more principled properties. like, lots of things vary together as you vary the size of an organism. diabetes presumably exists in various mammals, and has similar symptoms; you might be able to discover diabetes from this raw data

a theme:
* you could do this iteratively, improving your properties on each step?
	* eg you discover blood pressure, then you discover diabetes, then you discover diabetes comes in two kinds

sometimes you want to go and investigate what underlies some cluster. eg you might need to look into stuff to understand diabetes has to do with the pancreas malfunctioning
