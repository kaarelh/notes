some hyperparams of human concepting:
* concepts are components from which one constructs models to handle various situations one encounters
* the point of a model is to help one do the following (overlapping) things:
	* act well/effectively (e.g. get groceries, get admitted to a university, ban AGI)
	* determine some fact, determine the value of some property (e.g. you might want to know where your glasses are; in a physics problem you might want to what the friction force on an object is; you might be interested in whether P$\neq$NP or whether moral realism is right)
	* solve some problem (eg: construct a mathematical object with some properties, make a technology which allows people to quickly move between cities, write a program that sorts a list while only using a small amount of memory)
	* understand/learn something (e.g. you might want to become able to find solutions to diophantine equations in the future or come up with a clear way to think about the effect of greenhouse gas emissions on animal populations)
