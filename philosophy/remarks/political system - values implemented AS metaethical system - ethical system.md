
the title is to be read as an analogy

humanity has norms (\approx current values) and metanorms (\approx politics)

humanity's chosen metaethical system is politics/democracy i guess? humanity is pretty much rationally alright with that?

consider:
1: losing some amount of reflection ability (e.g. lose interest in ethics) to save oneself from 10 h of pain, or, maybe in more correspondence with what some people think is legit, just to make oneself happier
2: giving up some amount of democracy for a society where some policies go one's way (e.g. taxes, schooling, what kind of stuff the state uses people's money to fund). or maybe (for a better analogy) imagine a humanity-level thing like this, eg where we create a constitution that specifies that a bunch of object-level stuff needs to be the way it is currently, making it quite hard to change that in the future (imagine that you need an 80% majority for a change to the constitution)

1 and 2 feel like tradeoffs of a similar kind. in the first case, giving up personal metaethical ability for an object-level thing; in the second case, at least the second variant of the second case, giving up humanity's metaethical ability for an object-level thing
