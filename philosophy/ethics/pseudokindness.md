my first thought is that there's a very natural but very weak version of keeping all the weak agents around which is to simply keep all possible agents around which looks like implementing 'all possible small enough physical structures'/'running all turing machines in parallel' (with sth like dovetailing, ovetailing_(computer_science)#Infinitely_many_simultaneous_computations ). this is plausibly sorta guaranteed to happen in some not-too-unreasonable sense. unfortunately, this is obviously not great for any particular complicated thing like a human or humanity, because you'll get like 0 steps of us being run even with all resources in this universe being put toward this giant universal computation

anything more meaningful than this seems to require that some special care is given to agents that are already there in the universe. it seems sorta plausible to me that there are actually many agents that are equally already present 'in the universe' to the superintelligence, such as different reasonable developmental versions of itself, other close minds, and various other minds it thinks about or creates

anyway, assuming the ASI identifies humanity as among the top quadrillion agents which 'exist', then two things that seem pretty ASI-mind-independently natural are:
1) leaving humanity alone
2) making many copies of them and leaving those alone
