
Thm. For any finite group $G$, there is a point set $S\subseteq \mathbb{R}^n$ (for some $n$) such that $G$ is isomorphic to the group of automorphisms of $S$ — i.e., the group of isometries of $\mathbb{R}^n$ which send $S$ to $S$.

In fact, the proof below shows that any finite group $G$ is the group of isometries of a perturbed simplex.

Pf. Let's define the Cayley graph of $G$ to be the edge-colored graph whose vertices correspond to elements of $G$, with each vertex $g$ having, for each group element $s$, an $s$-colored edge to $gs$ — i.e., the colors correspond to elements of $G$ as well. Note that $G$ is isomorphic to the group of color-preserving graph automorphisms of this graph. This is because a bijection $\phi\colon G \to G$ is a color-preserving graph automorphism if and only if for each pair $g,s$, we have that the edge from $\phi(g)$ to $\phi(gs)$ is $s$-colored, which is true iff $\phi(gs)=\phi(g)s$ for all $g$ and $s$, which implies that $\phi(1s)=\phi(1)s$, which implies that a graph automorphism must be $g\mapsto hg$ for some fixed $h\in G$; conversely, any such map is indeed a color-preserving graph automorphism since then indeed $\phi(gs)=hgs=\phi(g)s$. So these two groups are in natural bijection as sets, and they obviously have compositions which respect that bijection, so we indeed have an isomorphism. Now, match the identity color with $0$ and the other colors arbitrarily with $|G|-1$ distinct real numbers between $1$ and $1+\varepsilon$.
