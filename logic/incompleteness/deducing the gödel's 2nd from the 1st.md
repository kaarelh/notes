
thm. if system consistent, then can't prove it is consistent

suppose could prove. then has provable p implies p. then can think about first sentence "this sentence is not provable", and be like "if false, then provable, then true — contradiction" and then conclude that the sentence is true, so it is not provable. but then it can also look at this argument in quotes, recognize it as a proof of the sentence, and so get that the sentence is false. so the system is inconsistent
