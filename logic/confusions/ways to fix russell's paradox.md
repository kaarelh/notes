
Here's a picture. Imagine all the mathematical objects — all the sets, say (it's common to make all mathematical objects be sets at the end of the day). Now surely you can pick out the ones which satisfy any kind of intelligible/concrete property: for instance, you should be able to pick out all the ones which contain the element $\emptyset$, or which contain at least $20$ elements, or which do not contain themselves. Now, again surely, this collection you've picked out should itself be a mathematical object — after all, it seems perfectly natural to consider, say, all the sets which contain the element $\emptyset$, or which contain at least $20$ elements, or which do not contain themselves, much like it seems natural to consider all the natural numbers divisible by $2$, or the natural numbers which can be written as the sum of two primes.
