
this isn't a direct response, but some considerations that i think are probably relevant + true/good:
* i think it could be reasonable to say either of the following about intelligence:
	* "there are very many interesting thinking-structures already in use, and very many more will come to be used in the future, and very many more (even comparably useful ones) will never be invented"
	* OR "there's nothing mysterious, just a bunch of stupid stuff"
(* i feel more like saying the former than the latter)
* but i doubt it makes sense to say "it's sorta just sth like SGD + stupid stuff". here are some things that are imo more central than sth like SGD:
	* tool use and tool-making
	* language
	* logic
	* thinking about how to think, and then starting to think that way
		* note one needn't be very aware that this is what one is doing. example 1: a mathematician coming up with a good definition. example 2: more generally, seeing a need to talk about something and coining a word for it. example 3: a philosopher trying to clarify/re-engineer a term, eg by seeing which definition would fit best into the ways one wants to speak using that term. example 4: trying to figure out how to improve peer review. example 5: noticing and resolving tensions
		* this overlaps with tool use and tool-making, because concepts are thinking-tools. it overlaps with language because human concepts are significantly linguistic — they fit in a language together, they only let us do big things because of fitting together in a language. (really, we can't even properly conceive of them / 'they aren't themselves' except in the context of a language or linguistic practices even more broadly. note that i'm not claiming that language is all the necessary context for concepts to even make sense — one probably needs a lot of additional context also)
		* if i had to pick one thing to say doing very much is all about, then maybe i'd say this
		* it's plausible this deserves to be split into many (sub)categories
	* compression
	* facing a curriculum of problems. creating problems for yourself to learn. play
	* keeping problems in mind as you go about your business, thinking about whether something you notice could be applied.
	* goal-pursuits. working on projects
	* seeing/making/using analogies
	* teaching/explaining
	* scale
	* probably https://www.lesswrong.com/posts/CFyt5fRpCQk8spDuR/what-are-the-primary-drivers-that-caused-selection-pressure?commentId=Yhr2ABiKAjQeJDwQi and https://tsvibt.github.io/theory/pages/bl_24_08_15_05_24_51_779476.html could motivate more things to add to this list
* one might object to some or all the above, saying these aren't responses to the right question. but what question are we asking? maybe: which combinations of structures make a process which goes on to do/understand very much? some issues with this question:
	* one could ask this with a cognitive system with some very fixed structure in mind. like, maybe you have a particular kind of 'world model' and particular other components, and voila, this thing goes on to understand arbitrarily much math/science by just 'filling in some content' into these structures. which structures make this possible?
		* i think this picture is basically confused, because to understand very much decently quickly, you probably have to keep reprogramming yourself on all levels forever. we could maybe imagine a mind which gets very far while 'keeping the same structure', but i think such a mind would just be building new structures that are not seen in the explicit structure you forced on it (e.g. imagine a system somehow forced into not using probabilities explicitly which effectively starts using probabilities anyway in some contrived way, because that's a useful thinking-structure). if we insist on no new structures really being incorporated, then imo the answer to "which structures make fooming possible?" is just "there are no such structures — fooming is not possible this way"
	* one could ask this allowing new structures to be formed. like, literally, which processes go on to do/understand very much?
		* are we happy with the answer 'for example, a sufficiently big evolution'? i feel like we're not happy with this answer despite it being true, so this question isn't exactly what we're meaning to ask. i do think it's a very interesting question though, about which very many interesting things could be said. i think there will be many very substantively different ways to get such a process, and it'd be cool to understand much more about which processes work
	* okay, maybe we wanted to ask instead: the formation/invention/discovery of which structure first makes such a process foom decently fast?
		* i think there are plausibly many structures that could be formed in various orders here that would increase the speed of fooming. i think all of the following structures have a reasonable claim to having contributed significantly to humanity fooming: replication/evolution (made fooming much faster than physics without evolution), vision, (large gap here because i got bored with adding to this list,) learning stuff from parents, opposable thumbs, manual dexterity, persistence hunting in groups, tool use, tool-making/maintenance/improvement, using fire, language, culture, trade, writing, governance, schooling, mathematical proof, the printing press, the scientific method, democracy, the use of statistics, medicine, first-order logic, computers, programming, the internet (and really basically any technology, way of thinking, way of acting, concept, but these are some central ones)
		* i think the speed of fooming will be increased by many structures in the future, plausibly arbitrarily many times (until the end of time or whatever), and plausibly even including arbitrarily many new structures as profound as developing primitive language
		* this kind of thing doesn't seem like an entirely satisfying answer either
	* okay, maybe we only meant the structures which are in use when a current individual human paces around, temporarily doesn't interact with anything external, and tries to make research progress?
		* i think very many interesting structures are in use. i've mentioned some above
		* eg each word has some little structures associated with it
		* maybe we're asking for some higher structure organizing these little structures?
			* i think there's a major aspect of: the context for each little structure is created by all the other little structures, not something external to these little structures
			* anyway, the external thing will also have a lot of rich structure
			* if i had to pick sth short to say about a separate higher organizing thing, i'd probably say (1) 'effectively, we're thinking about how to think and then thinking that way', (2) our concepts have close shadows in language. and then maybe next i'd want to say a bunch more about language
			* maybe this is still too high-level? but idk, are we interested in neuronal weight update rules?
