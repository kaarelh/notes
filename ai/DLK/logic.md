
If a model gives an assignment of truth to sentences which is stable under inference rules, then the set of all sentences assigned to be true by deductions is consistent, I think. And then by Gödel's completeness theorem as stated here https://plato.stanford.edu/entries/logic-classical/, "Let $\Gamma$ be a set of sentences. If $\Gamma$ is consistent, then $\Gamma$ is satisfiable.", we have that the set of statements is also satisfiable, meaning that there is a model of the theory satisfying it. I think this is an argument that any assignment of truth-values to sentences which does not violate inference rules can be extended to a full model (this is cool because a model also needs to assign actual functions and objects and stuff to the corresponding variables).
