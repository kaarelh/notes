In addition to vanilla solomonoff induction, there are some other kinds of ideal induction I reference when thinking about AI-relevant (philosophical) questions. I want it to be easier to use these notions when talking to others (e.g. when talking to others via essays), so I’m publishing some short notes introducing the notions and stating some fairly obvious theorems involving them.
* functional solomonoff induction
* positive example induction
* solomonoff axiom induction
* property induction
* circuit induction
* time-bounded universal induction
