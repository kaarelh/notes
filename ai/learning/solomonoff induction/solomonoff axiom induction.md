# other ideas for how to fix axiom induction

* i think we probably don't want to ask for precise answers to all questions
	* we should be fine with only generating a very small subset of observations
	* we should be fine with making some mistakes. like, the schema should probably be allowed to prove some sentences marked false
	* if we have any facts stated as "this variable has this value", then we should have a metric on values grade generated sentences. like, you shouldn't have to generate precisely right sentences; it's already great if you get close
