* an example from wentworth:
	* the problem is to give a turing machine that doesn't halt
	* this is very easy to solve yourself: eg just write down an infinite while loop
	* but it is really really hard to give a verifier here — there is no algorithm solving the halting problem
	* well, but we could also just ask for a TM together with a proof that it doesn't halt; this will in fact be easy to verify
	* also, it's not hard to provide a proof here
	* so in some sense, this isn't a compelling counterexample to the verification hope
