# How do we keep regulating fooming?
* well, we ban AI for now, and then we ban it harder later when that becomes necessary, and then we ban it even harder if that becomes necessary, and so on potentially forever or however long making an alien top thinker remains a bad idea. at some point, we might have ideas, methods, institutions, technologies that make it pretty easy to not make any AI smarter than us, maybe sorta like it would be easy for an individual to decide not to make an AGI and then to simply not make an AGI for a very long time (not forgetting this commitment, not getting confused and accidentally carrying out a motor sequence that)
* regulating fooming is probably going to happen even with an AI, *
* more generally, we should make it so technologies are more human-chosen, not just universally dropped into the world as soon as developed. though most technologies are beneficial, there is no universal law that says technologies should always be introduced when invented. we need to make it possible for society to be deliberate about what technologies to introduce when, and in particular about the order in which technologies are introduced. we need to set up institutions that are responsible for doing this and able to do it even for novel technologies and ahead of time, though such institutions should use their powers sparingly

# We can go extremely far (and fast) as humans

* We can probably get to each intelligence level with human-descended systems (modulo not running out of negentropy, and if we want to), it'll just take somewhat longer. It's really just a different path to arbitrarily capable systems.
* Actually, it might not even take us that much longer to foom to any given intelligence level, because a misaligned AGI might also want to self-improve carefully instead of recklessly fooming at full possible speed. So it might only take us longer by however long it takes us to get to a position where our fooming option space is similar to the option space of that misaligned AGI. So there might be a small (on galactic timescales) constant time offset T such that for any intelligence level X, a misaligned AGI would foom to level X only T sooner than us. (There are probably important corrections to this model but I think this is a decent zeroth order analysis of the situation.)
* So, we can also develop extremely fast as humans (if we want to), to the extent that a fooming AI would.

# Conclusions

My prescription is not that we should commit to never making it so that the top thinker is a highly non-human-descended AI. We should build systems that make it possible for humanity to make momentous decisions thoughtfully. For now, we should be on a foom that is properly our own, and so we should ban AI. Later, after having grown much more capable and wise and having thought about intelligence, learning, values etc very much more, humanity might thoughtfully decide to unban AI and create a non-human top thinker, or might decide not to do that.
